# :iphone: Mapli
__나의 트랙리스트를 앱에 준비된 바탕 템플릿들에 담아 보이는 앱입니다.__

__Apple Music과의 연동을 통해 별도 로그인 없이 나의 트랙리스트를 불러올 수 있습니다.__

__앱에 기본적으로 준비된 템플릿들은 별도 결제없이 모두 사용 가능합니다.__
